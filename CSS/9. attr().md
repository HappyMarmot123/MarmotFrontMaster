# CSS attr() 함수

## 📖 개요

**CSS attr() 함수**는 HTML 요소의 속성(attribute) 값을 CSS에서 직접 사용할 수 있게 해주는 강력한 기능입니다. 이 함수를 통해 동적인 콘텐츠 생성, 반응형 디자인, 그리고 HTML과 CSS 간의 효율적인 데이터 교환이 가능합니다.

**attr() 함수가 현대 웹 개발에서 중요한 이유:**

- **동적 콘텐츠**: HTML 속성 값을 CSS 콘텐츠로 동적 표시
- **코드 중복 제거**: 하드코딩된 텍스트 대신 속성값 활용
- **유지보수성**: HTML에서 값 변경 시 CSS 자동 반영
- **접근성**: 의미 있는 HTML 속성을 시각적으로 표현
- **국제화**: 다국어 지원을 위한 효율적인 구조

**attr() 함수의 핵심 가치:**
attr() 함수는 HTML의 의미적 구조와 CSS의 표현 능력을 연결하는 다리 역할을 합니다. 이를 통해 개발자는 중복 코드를 줄이고, 더 유지보수하기 쉬운 코드를 작성할 수 있으며, 특히 도구 설명, 라벨, 상태 표시 등에서 강력한 효과를 발휘합니다.

## 🎯 attr() 함수 기본 문법

### 1. **기본 사용법**

**attr() 함수의 구문:**

```css
attr(attribute-name)
attr(attribute-name, fallback-value)
```

**기본 활용 패턴:**

```css
/* HTML 속성 값을 콘텐츠로 사용 */
.tooltip::after {
  content: attr(data-tooltip);
}

/* 폴백 값 지정 */
.label::before {
  content: attr(data-label, "기본값");
}

/* 여러 속성 조합 */
.user-info::after {
  content: attr(data-name) " (" attr(data-role) ")";
}
```

**실무 활용:**

```css
/* 기본 툴팁 구현 */
.tooltip {
  position: relative;
  cursor: help;
  border-bottom: 1px dotted #6b7280;
}

.tooltip::after {
  content: attr(data-tooltip);
  position: absolute;
  bottom: 100%;
  left: 50%;
  transform: translateX(-50%);
  background-color: #1f2937;
  color: white;
  padding: 8px 12px;
  border-radius: 6px;
  font-size: 0.875rem;
  white-space: nowrap;
  opacity: 0;
  visibility: hidden;
  transition: opacity 0.3s, visibility 0.3s;
  z-index: 1000;
  margin-bottom: 8px;
}

.tooltip:hover::after {
  opacity: 1;
  visibility: visible;
}

/* 상태 표시 */
.status-badge {
  position: relative;
  padding: 4px 8px;
  border-radius: 4px;
  font-size: 0.75rem;
  font-weight: 500;
}

.status-badge::before {
  content: attr(data-status);
  text-transform: uppercase;
}

.status-badge[data-status="active"] {
  background-color: #dcfce7;
  color: #166534;
}

.status-badge[data-status="pending"] {
  background-color: #fef3c7;
  color: #92400e;
}

.status-badge[data-status="inactive"] {
  background-color: #fee2e2;
  color: #991b1b;
}
```

### 2. **HTML 표준 속성 활용**

**표준 속성의 효과적 사용:**

- title, alt, href 등 표준 속성 활용
- 접근성 정보의 시각적 표현
- SEO 친화적인 구조 유지

**실무 활용:**

```css
/* 링크 URL 표시 */
.external-link {
  position: relative;
  color: #3b82f6;
  text-decoration: none;
}

.external-link::after {
  content: " (" attr(href) ")";
  font-size: 0.75rem;
  color: #6b7280;
  font-weight: normal;
}

/* 인쇄용 스타일에서 링크 URL 표시 */
@media print {
  a[href]::after {
    content: " (" attr(href) ")";
    font-size: 0.8em;
    color: #666;
  }
}

/* 이미지 alt 텍스트 표시 (접근성) */
img[alt]::after {
  content: attr(alt);
  display: block;
  font-size: 0.875rem;
  color: #6b7280;
  font-style: italic;
  margin-top: 4px;
  padding: 4px;
  background-color: #f9fafb;
  border-radius: 4px;
}

/* 약어 설명 표시 */
abbr[title] {
  border-bottom: 1px dotted #6b7280;
  cursor: help;
}

abbr[title]::after {
  content: " (" attr(title) ")";
  font-size: 0.75rem;
  color: #6b7280;
  font-weight: normal;
}

/* 입력 필드 플레이스홀더 스타일링 */
.custom-input {
  position: relative;
}

.custom-input input:placeholder-shown + .floating-label::after {
  content: attr(data-placeholder);
  position: absolute;
  top: 50%;
  left: 12px;
  transform: translateY(-50%);
  color: #9ca3af;
  pointer-events: none;
  transition: all 0.2s ease;
}
```

## 🏷️ 커스텀 데이터 속성 활용

### 1. **data-\* 속성과 attr() 조합**

**데이터 속성의 장점:**

- 의미적으로 명확한 정보 저장
- JavaScript와 CSS 간 데이터 공유
- 유효한 HTML 마크업 유지

**실무 활용:**

```css

  content: "";

  top: 0;
  left: 0;

  top: 50%;
  left: 50%;
```

## 📱 반응형 및 접근성

### 1. **반응형 attr() 활용**

**화면 크기별 적응형 콘텐츠:**

- 모바일에서 축약된 텍스트
- 데스크톱에서 상세 정보
- 컨텍스트에 맞는 정보 표시

**실무 활용:**

```css
/* 반응형 라벨 */
.responsive-label::after {
  content: attr(data-label-short);
  font-size: 0.875rem;
  color: #6b7280;
}

/* 태블릿 이상에서는 전체 라벨 표시 */
@media (min-width: 768px) {
  .responsive-label::after {
    content: attr(data-label-full);
  }
}

/* 모바일에서는 아이콘만 표시 */
.action-button::after {
  content: attr(data-icon);
  font-size: 1.25rem;
}

/* 데스크톱에서는 텍스트도 함께 표시 */
@media (min-width: 1024px) {
  .action-button::after {
    content: attr(data-icon) " " attr(data-text);
  }
}

/* 터치 디바이스용 확장된 툴팁 */
@media (hover: none) and (pointer: coarse) {
  .touch-tooltip::after {
    content: attr(data-tooltip-mobile);
    position: relative;
    display: block;
    background: none;
    color: #6b7280;
    font-size: 0.875rem;
    padding: 4px 0;
    margin-top: 4px;
  }
}
```

## 🎯 결론

CSS attr() 함수는 HTML과 CSS 간의 효율적인 데이터 교환을 가능하게 하는 강력한 도구입니다. 이를 통해 중복 코드를 줄이고, 유지보수성을 향상시키며, 동적이고 반응형인 웹 인터페이스를 구축할 수 있습니다.

**성공적인 attr() 함수 활용을 위한 핵심 요소:**

1. **의미적 HTML**: 표준 속성과 data-\* 속성의 적절한 활용
2. **동적 콘텐츠**: 속성값에 기반한 조건부 스타일링
3. **국제화 지원**: 다국어 환경을 고려한 콘텐츠 구조
4. **접근성**: 보조 기술과 스크린 리더 지원
5. **반응형 설계**: 디바이스별 최적화된 정보 표시

attr() 함수를 마스터하면 더 유연하고 유지보수하기 쉬운 CSS 코드를 작성할 수 있으며, HTML의 의미적 구조를 활용한 효율적인 스타일링이 가능합니다.
